# rell-health

> RELL instance for healthcare data standards and compliance.

**License:** BSL  
**Status:** Planned | Phase 4

## Target Market

Hospitals, health insurers, healthcare data vendors, EHR providers.

## Profiles

HIPAA Privacy + Security Rules, HITECH Act, HL7 FHIR R4

## Why This Market

HIPAA violations carry penalties of $100–$50,000 per violation, up to $1.9M per year per violation category. Healthcare organizations have budget for compliance tools because the cost of non-compliance is existential.
